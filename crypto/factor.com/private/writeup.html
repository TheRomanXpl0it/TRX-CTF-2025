This challenge is similar to standard RSA, except that the modulus NNN is a product of multiple primes, some of which can be very small.

Exploiting the Structure

Instead of waiting for all primes to be small (which could take an impractical amount of time), we can extract small pieces of information at a time by factoring parts of NNN.

If we manage to extract even one small prime factor qqq of NNN, we can immediately start recovering the flag:

Breaking It Down

We are given the standard RSA equation:

flagemod  N=c \text{flag}^e \mod N = c flagemodN=c

If we find a small prime qqq that divides NNN, we can reduce the equation modulo qqq:

cmod  q=(flagemod  N)mod  q=flagemod  q c \mod q = (\text{flag}^e \mod N) \mod q = \text{flag}^e \mod q cmodq=(flagemodN)modq=flagemodq

Since eee is known, we can compute the eee-th root modulo qqq to recover:

flagmod  q \text{flag} \mod q flagmodq

Reconstructing the Full Flag

By repeating this process for multiple small prime factors, we collect a system of modular equations:

flagmod  q1,flagmod  q2,…,flagmod  qn \text{flag} \mod q_1, \quad \text{flag} \mod q_2, \quad \dots, \quad \text{flag} \mod q_n flagmodq1​,flagmodq2​,…,flagmodqn​

Finally, using the Chinese Remainder Theorem (CRT), we reconstruct the original flag.
